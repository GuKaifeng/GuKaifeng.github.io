JavaBean

发表于2020-03-16|更新于2021-01-01|关于编程语言的基础知识学习笔记
|阅读量:|评论数:

JavaBean指符合如下标准的 Java 类：

  1. 类是公共的；
  2. 有一个无参的公共构造器；
  3. 有属性，属性一般私有，且有对应的 get()，set() 方法
